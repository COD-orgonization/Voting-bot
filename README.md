# Voits-bot

# Что престоит сделть
1. Создание бд которая будет получать и выдоваь даные
2. Реализация работы с пользоваелем с вожможносьтю разделения ролей (админ и пользователь)

## Что доступно всем
1. Возможнось просматривать и голосовать за учасников которые зарегистрировались
2. Быстро посматривать всех учасников можно с помощью списка
3. Регистрация как учасник

## Что доступно админу
1. Начинать и завершать голосование
2. Блокировать пользователей

## Регистрация включает
1. Ввод ФИО
2. Ввод описания "О себе"
3. Ввод фотографии
